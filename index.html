Editor Tools for Projects

Level Editor Tool 

I made this editor tool as a level design interface.

This allows me to read/write current level layout, including background, enemies, player spawn point, and  obstacles into data containers.   

Equipment Editor 

This tool can read/write item prefab transform details from/into data containers,  on different display scenarios: icon/on-character/detail-panel.

It also remembers which hand the weapon is in, and if it's a helmet etc. automatically, allowing fast management of a vast library of items.
